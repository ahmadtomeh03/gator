# Gator 🐊 RSS Feed Aggregator

Gator is a CLI application written in **TypeScript** that allows users to subscribe to RSS feeds and view updates directly from the terminal.

The application stores feeds and user subscriptions in a **PostgreSQL database** and allows multiple users to follow different feeds.

---

# Requirements

To run this project you need:

- Node.js (v18+ recommended)
- PostgreSQL
- npm
- drizzle-kit
